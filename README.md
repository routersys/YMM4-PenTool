# 拡張ペンツール for YMM4

[![License: LGPL v3](https://img.shields.io/badge/License-LGPL_v3-blue.svg)](https://www.gnu.org/licenses/lgpl-3.0)
[![.NET](https://img.shields.io/badge/.NET-10.0-purple.svg)](#)
[![Release](https://img.shields.io/github/v/release/routersys/YMM4-PenTool.svg)](https://github.com/routersys/YMM4-PenTool/releases)

YukkuriMovieMaker4 (YMM4) 上で動作する、高機能・高品質な**ペン描画図形プラグイン**です。  
ペン・マーカー・鉛筆・消しゴムの4種類のブラシと、レイヤー管理システム・操作履歴・直感的なキャンバス操作を備え、映像制作における手書き描画の表現力を大幅に向上させます。  
Direct2D によるハードウェアアクセラレーションレンダリング、完全なアニメーション統合、そして自由に配置・リサイズ可能なフローティングパネルUIにより、プロフェッショナルな制作環境を提供します。

![image](https://github.com/routersys/YMM4-PenTool/blob/main/docs/PenTool.png)

---

## 概要

このプラグインは、YMM4 に高機能なペン描画ツールを追加する図形プラグインです。  
標準機能から一歩進んだレイヤー管理・筆圧感知・豊富な操作系を備えており、複雑な手書き表現やアニメーション演出を直感的に作成できます。  
専用の編集ウィンドウは独立したフローティングパネルで構成されており、作業スタイルに合わせてレイアウトを自由にカスタマイズできます。  
また、YMM4 のアニメーションシステムと完全に統合されているため、太さ・長さ・オフセットにキーフレームアニメーションを設定することで、描線の成長アニメーションや動的な演出が実現できます。

---

## 主な機能

### 1. 高機能ブラシシステム
- **4種類のブラシ**: ペン・マーカー・鉛筆・消しゴムを用意。用途に応じて切り替え可能です。
- **ペン**: 標準的な描画に最適な丸ペン。色・太さ・筆圧感知を個別に設定できます。
- **マーカー**: 半透明の重ね塗りに対応した矩形チップの蛍光マーカー。
- **鉛筆**: 粗めのタッチを表現できる鉛筆ツール。デフォルトはグレー・細めに設定されています。
- **消しゴム**: ポイント消し（部分消去）とストローク消し（1本単位の消去）の2モードに対応。
- **筆圧感知**: 各ブラシで筆圧のON/OFFを個別に設定できます。
- **カラーピッカー**: 描画色をリアルタイムに変更可能です。

### 2. レイヤーシステム
- **複数レイヤー管理**: 複数のレイヤーで描画を分けて管理できます。
- **表示/非表示の切り替え**: 目のアイコンで各レイヤーの表示状態を制御します。
- **ロック機能**: レイヤーをロックして誤編集を防止します。
- **不透明度**: レイヤーごとに不透明度（0〜100%）を設定できます。
- **並び替え**: レイヤーのドラッグ＆ドロップによる直感的な順序変更に対応しています。
- **レイヤーサムネイル**: 各レイヤーの内容がリアルタイムにサムネイル表示されます。ホバー時にプレビューも確認できます。
- **自動ナンバリング**: レイヤーは追加・削除・並び替えに合わせて自動的に番号が更新されます。

### 3. 操作履歴（アンドゥ・リドゥ）
- 描画・消去・選択・レイヤー追加/削除/移動/プロパティ変更など、すべての操作が履歴に記録されます。
- `Ctrl+Z` / `Ctrl+Y` / `Ctrl+Shift+Z` でアンドゥ・リドゥが実行できます。
- 履歴パネルから任意の状態に直接ジャンプすることも可能です。

### 4. インタラクティブなキャンバス操作
- **ジョイスティックによるパン**: キャンバス操作パネルのジョイスティックをドラッグしてキャンバスをスムーズにスクロールできます。
- **マウスホイール操作**: マウスホイールにズームとペンサイズ変更のどちらを割り当てるか設定できます。
- **中ボタンパン**: マウスの中ボタンドラッグでキャンバスをパンできます。
- **ズームコントロール**: ズームスライダーおよびズームイン・アウトボタンで拡大率（0.1〜10倍）を調整できます。
- **回転ノブ**: キャンバス操作パネルの外周ノブをドラッグしてキャンバスを任意の角度に回転できます。
- **フィット表示**: ボタン一つでキャンバス全体をビューポートに収めるフィット表示を実行できます。

### 5. 選択・変形ツール
- **選択モード**: キャンバス上のストロークを矩形選択して操作できます。
- **移動**: 選択したストロークをドラッグで移動できます。
- **リサイズ**: 選択ハンドルで選択範囲のサイズを変更できます。
- **回転**: 選択範囲の上部に表示される回転ノブでストロークを自由に回転できます。
- **コントラスト対応の選択枠**: 背景色の明度を自動判定し、選択枠の色（白/黒）を切り替えることで視認性を維持します。
- **レイヤー制限**: 選択操作は現在選択中のレイヤーに限定されます。

### 6. アニメーション完全統合
- **Thickness（太さ）**: 描画全体の太さをアニメーションで変化させられます（0.1〜200%）。
- **Length（長さ）**: 描線の始点から終点まで描画する割合をアニメーションで制御し、描画アニメーションを実現します（0〜100%）。
- **Offset（オフセット）**: 描画開始位置をずらすことで、ループアニメーションや特殊演出が可能です。
- すべてのパラメータは YMM4 のタイムライン上でキーフレームアニメーションを設定できます。

### 7. 自由なUIレイアウト
- 描画キャンバス・レイヤーパネル・キャンバス操作パネル・履歴パネルの4つのパネルが独立したフローティングウィンドウとして機能します。
- 各パネルはヘッダーをドラッグして自由に移動でき、8方向のリサイズハンドルでサイズ変更も可能です。
- スナップ機能により、パネル同士の端や画面端に吸い付くように整列できます。
- **最前面固定**: パネルごとに「常に最前面」表示を設定できます。
- **半透明**: パネルを半透明化して背後のキャンバスを透かして見ることができます。
- **レイアウト永続化**: パネルの位置・サイズ・表示状態は設定に自動保存されます。
- **ツールバー配置**: ツールバーを上・下・左・右の4箇所から選択できます。

### 8. インポート / エクスポート
- **PNG エクスポート**: 全レイヤーを合成した描画を PNG 画像として保存できます。
- **ISF エクスポート**: 選択中のレイヤーを Ink Serialized Format（`.isf`）形式でエクスポートできます。
- **ISF インポート**: `.isf` ファイルを現在のレイヤーに読み込むことができます。

### 9. アップデート通知
- 起動時に GitHub の最新リリースを確認し、新しいバージョンが利用可能な場合に通知します。
- 「いいえ」を選択すると、そのバージョンの通知は以降表示されなくなります。

### 10. 多言語対応
- 日本語・英語・簡体字中国語・繁体字中国語・韓国語・スペイン語・アラビア語・インドネシア語の8言語に対応しています。

## 動作要件

- **OS**: Windows 10 / 11 (64bit)
- **YukkuriMovieMaker4**: 最新版を推奨
- **ランタイム**: .NET 10.0

---

## インストール方法

1. **[リリースページ](https://github.com/routersys/YMM4-PenTool/releases/latest)** から最新の `.ymme` ファイルをダウンロードします。
2. YMM4 が起動していないことを確認し、ダウンロードしたファイルをダブルクリックしてインストールします。
3. YMM4 を起動します。

## 使い方

### 基本的な使用フロー

1. YMM4 のタイムライン上で右クリックし、「図形」を選択します。
2. 設定パネルの「種類」から **「拡張ペンツール」** を選択します。
3. プロパティパネルに表示される **「ペンツールで編集」** ボタンをクリックすると、専用の編集ウィンドウが開きます。
4. ツールバーからブラシを選択し、キャンバス上に描画します。
5. ウィンドウを閉じると描画内容が YMM4 のアイテムに反映されます。

### ツールバー操作一覧

| 操作 | 動作 |
|---|---|
| ペンアイコン | ペンブラシを選択（右クリックで筆圧ON/OFF） |
| マーカーアイコン | マーカーブラシを選択（右クリックで筆圧ON/OFF） |
| 鉛筆アイコン | 鉛筆ブラシを選択（右クリックで筆圧ON/OFF） |
| 消しゴムアイコン | 消しゴムを選択（右クリックでポイント/ストローク消しを切替） |
| 選択アイコン | 選択モードに切り替え |
| アンドゥ | 直前の操作を取り消す（長押しで連続実行） |
| リドゥ | 取り消した操作をやり直す（長押しで連続実行） |
| カラーピッカー | 描画色を変更 |
| 太さスライダー | ブラシの太さを変更（0.1〜1024） |
| インポートアイコン | ISF ファイルをインポート |
| エクスポートアイコン | PNG または ISF 形式でエクスポート |

### キーボードショートカット

| キー | 動作 |
|---|---|
| `Ctrl + Z` | アンドゥ |
| `Ctrl + Y` | リドゥ |
| `Ctrl + Shift + Z` | リドゥ（代替） |

### キャンバス操作一覧

| 操作 | 動作 |
|---|---|
| マウスホイール | ズームまたはペンサイズ変更（設定により切替） |
| 中ボタンドラッグ | キャンバスをパン |
| ジョイスティックドラッグ | キャンバスをスクロール |
| 外周ノブドラッグ | キャンバスを回転 |
| フィットボタン | キャンバス全体をビューポートに収める |
| リセット回転ボタン | キャンバスの回転をリセット |

### レイヤー操作一覧

| 操作 | 動作 |
|---|---|
| 目のアイコンクリック | レイヤーの表示/非表示を切替（履歴に記録） |
| レイヤー名ダブルクリック | レイヤー名を編集 |
| ドラッグ＆ドロップ | レイヤーの順序を変更 |
| ⋮ アイコンクリック | レイヤープロパティ（不透明度）を編集 |
| ＋ ボタン | レイヤーを追加 |
| − ボタン | 選択中のレイヤーを削除 |
| ↑ / ↓ ボタン | レイヤーを上/下に移動 |

### アニメーション設定

YMM4 のプロパティパネルにある以下のパラメータにアニメーションを設定することで、描画を動的に演出できます。

- **太さ**: 値が大きいほど描線が太くなります（基準値100%）。
- **長さ**: 0% で描画なし、100% で全体を表示。増加させることで描線が成長するアニメーションを作れます。
- **オフセット**: 描画の開始点をずらします。アニメーションと組み合わせることで巡回表示が実現できます。

## 注意事項

- 本プラグインを使用する前に、YMM4 プロジェクトファイルのバックアップを作成することを強く推奨します。
- 描画データはプロジェクトファイル内に保存されます。プリセット等の外部ファイルは存在しません。
- ロックされたレイヤーへの描画・消去は自動的にキャンセルされます。
- インターネット通信はアップデート確認のみに使用します。アップデート確認の通信は GitHub API（`api.github.com`）に対してのみ行われます。
